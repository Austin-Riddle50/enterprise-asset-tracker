# enterprise-asset-tracker

# Overview
Enterprise asset tracker is a java-based web app designed to manager and audit internal organizational assets.
The system supports role-based access and maintains an audit trail for asset changes, reflecting enterprise and government-contractor application requirements.

## Technology Stack
- Java (Jakarta EE)
- Enterprise Java Application Server (WildFly / Apache EAP)
- PostgreSQL
- Linux environment (WSL)

Project initialization and environment setup in progress.
